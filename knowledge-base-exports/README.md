# Knowledge Base Exports

Auto-generated KB artifacts exported from the knowledge-base service. These are NOT decision proposals.

## Why this directory exists

On 2026-04-16, as part of token-optimization Tier 1 task A2, 76 files were moved here from `~/environment/proposals/`. They had filenames matching `2026-04-14-kb-item-*`, `2026-04-14-kb-raise-*`, `2026-04-14-kb-test-*`, and `2026-04-14-kb-history-*`.

These auto-generated KB export files were polluting the `proposals/` directory (which is scanned by agents for decision proposals) and adding ~22k tokens per scan for no value.

## What these files are

Automated exports generated by the knowledge-base service during testing and bulk import operations on 2026-04-14. Each file represents a KB item that was raised or stored via the KB service API.

## Rules for this directory

- Files here are archived exports — do NOT edit them
- Do NOT scan this directory for proposals or decisions
- If you need to reference a specific KB item, read it directly by filename
- New KB exports should also go here, not in proposals/
