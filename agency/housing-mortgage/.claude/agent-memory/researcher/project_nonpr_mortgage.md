Researched 2026-04-11. Full findings in .worklog/non-pr-mortgage-research.md.

Key facts:
- Cash purchase is the most viable path at ¥5.65M (price point matches minimum loan floors)
- Tokyo Star Bank Star Mortgage: only purpose-built non-PR product. Min loan ¥5M, income ¥3-4M+, Japanese literacy required, urban property bias
- Nanto Bank: ¥2M income floor, NO explicit PR requirement in published specs — requires direct inquiry (most promising unknown)
- Asuka Credit Union (あすか信用組合): published non-PR foreign housing loan for Nara region — NOT fully investigated yet
- Megabanks (MUFG, Mizuho, SMBC direct), Flat 35, Aruhi: require PR — closed
- Prestia SMBC: non-PR OK but ¥5M income floor + 2.2% admin fee
- PR spouse co-borrower: best mortgage path if available (Flat 35 via spouse)
- FEFTA Form 22: Japan-resident foreigners with valid zairyu card are EXEMPT (classified as residents)
- Timeline: 8-12 weeks (non-PR) vs 6-8 weeks (PR)
- Rate premium: 0.7-2.2% above baseline; total lifetime premium ~¥1.5-2.3M on ¥5.65M loan
- Nanto Bank inquiry question (Japanese): 外国籍で永住権を持っていない場合、住宅ローンの申込みは可能ですか？在留カードは持っています。

**Why:** Advisory for non-PR foreign buyer targeting Nara/Kyoto akiya.
**How to apply:** Use as baseline for any mortgage-path advice to non-PR clients. Asuka Credit Union and Nanto Bank need direct inquiry as next step.
