Researched 2026-04-08. Full findings returned to parent agent as structured text.

Key facts established:
- 60–90 day timeline end-to-end
- Non-PR mortgage: 20–50% down, 1–3yr Japan employment, visa 3+yr remaining
- Flat35 requires permanent residency
- Foreign-friendly lenders: Prestia SMBC, Tokyo Star Bank, Aeon Bank, Suruga Bank
- Nara English agents: Move2Japan (Kansai), Century21 Japan multilingual, Akiya Japan+teritoru, realestate.co.jp
- Dual agency legal but conflicted — buyers should engage own agent
- No separate buyer-side commission — agent earns from transaction pool
- Biggest pitfalls: all-Japanese docs, deferred acquisition tax, condo reserve deficits, akiya rebuilding restrictions
