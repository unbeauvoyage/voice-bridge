# Agency: Vending Machine Route — Kansai Japan

See `~/.claude/CLAUDE-common-to-all-agency-projects.md` for common rules.

## Mission
Build an actionable business model for a vending machine route in Kansai (Nara, Osaka, Kyoto, Wakayama). TYPE A passive: 3–5 machines, ¥100K–¥150K/month costs → ¥200K–¥300K revenue.

## Key Research Areas
- Monthly earnings per machine by location type (tourist, station, office, hospital)
- Location host revenue split (typical 10-20%)
- New vs used machine costs; leasing vs buying; Yahoo Auctions sources
- Placement contracts: securing tourist locations (Nara Park, Todai-ji, Yoshino)
- GASHAPON ANGLE: capsule toy machines at tourist spots (¥30K–¥100K/month, near-zero maintenance)
- Legal: food hygiene license, foreigner ownership under Business Manager Visa
- TYPE C: retiring operators selling routes below replacement value
