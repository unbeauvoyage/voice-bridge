Core finding: ¥150K/month cost estimate only holds for owned-property + self-cleaning + fully-amortized machines. Any leased Kansai property runs ¥250K–¥370K/month costs before loan repayment.

Revenue model validated: 10-machine store at 10% utilization generates ~¥504K/month. ¥300K–¥500K revenue target is achievable at average performance.

**Why:** The CEO's baseline cost estimate was based on an optimistic owned-land scenario. Research confirmed that utilities alone (gas dryers ¥100K–¥150K + electricity ¥40K–¥50K + water ¥20K–¥30K) total ¥155K–¥230K, before adding rent and staffing.

**How to apply:** When CEO asks about profitability, always distinguish owned-land vs leased-property scenarios. ¥100K–¥200K net monthly is the realistic expectation for Nara/Wakayama leased 10-machine stores.

## Alternative Path Research (Wave 4, 2026-04-09)
- ¥1M-1.5M alone is NOT enough for a standalone new store (smallest new build: ¥18.4M). Viable as acquisition price of distressed Kansai stores (Tranbi listings: Kashihara ¥1.5M, Osaka City ¥1.9M).
- Hotel partnership model confirmed: operator installs machines, hotel gets 40-50% of gross revenue, operator keeps 50-60%. Yield per hotel: 6-10% on ¥2M-3M invested — supplemental only.
- Apartment building model confirmed: revenue share 30-60% to property owner. Operator earns ¥25K-60K/month per 2-4 machine set installed in apartment common area.
- Mini-laundry (5 machines, used equipment + 居抜き space): ¥3M-7M total. Net ¥50K-150K/month. Viable as first step.
- Container laundry: no ¥1M-2M product exists. Smallest confirmed is ¥20M-25M (COMOREBI). Advantage is mobility and no building permit, not cost reduction.
- Dry cleaning drop-off combo: no major chain has a public laundromat partnership program. Drop-box model earns ¥10K-30K/month. License required only if you actually clean on-premises.
- Vending + lockers add-on: +¥20K-70K/month. 3rd-party vending is zero capital; own lockers cost ¥80K-200K.
- New build vs acquisition comparison: acquisition wins on every metric. 居抜き + used machine hybrid: ¥11M-15.5M vs ¥21M-35M new build.

### Lease Intel — Confirmed Kansai Commercial Norms
- 保証金 (Kansai deposit): 5–10 months upfront; 2–3 months non-returnable (敷引き)
- NO renewal fee (更新料) in Kansai (unlike Tokyo)
- 定期借家 (fixed-term): AVOID — no automatic renewal, destroys infrastructure investment value
- All utilities are TENANT responsibility — confirmed
- Lease transfer requires written landlord consent — must obtain BEFORE making offer
