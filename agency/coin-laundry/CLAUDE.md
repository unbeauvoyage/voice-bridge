# Agency: Coin Laundry — Kansai Japan

See `~/.claude/CLAUDE-common-to-all-agency-projects.md` for common rules.

## Mission
Build an actionable business model for a coin laundry (コインランドリー) in Kansai (Nara, Osaka suburbs, Wakayama, Kyoto). TYPE A/B passive: ¥150K–¥200K/month costs → ¥300K–¥500K revenue.

## Key Research Areas
- Revenue for 10-machine laundromat in Nara/Osaka/Wakayama; break-even timeline
- Franchise vs independent: Wash House, Coinex, Tosei, White Laundry (fees, support)
- Used commercial machine prices; minimum viable machine count
- Location demographics: students, families, no-washer households
- TYPE C: laundries for sale (tenpo.com, gyoumu-tenpo.com, j-net21)
- Legal: licenses, fire safety for dryers, zoning, Business Manager Visa
